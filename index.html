Webpack

This repository is a tamplate. Webpack 5 is used as a bundler. To use this template, follow these steps:

  • - Replace the current files in the 'src' folder with those you need
  • - Replace the paths to the home page in the 'package.json' file on lines 4 and 14, substituting your values.
  • - Install additional packages you need using the command npm install or npm i
